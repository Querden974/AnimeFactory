AnimeFactory will give you updates on the week anime schedule and allow you to read informations about different animes.

## API Used:
[Anilist API](https://docs.anilist.co/)  used to fetch animes data.

## Stack

-   Laravel
-   Blade
-   Tailwind
-   Javascript
-   mySQL
-   GraphQL
